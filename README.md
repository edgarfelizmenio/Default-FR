# Default-FR
Default Facilities Registry

## Setup
1. Edit the following files:
    * ```nginx/default-fr-nginx```
    * ```nginx/default-fr-nginx.conf```
    * ```upstart/default-fr-upstart.conf```
    * ```config.py```
    * ```install.sh```
2. Run ```install.sh```
    ```
    ./install.sh
    ```
